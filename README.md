# dr-igor.github.io

Pages site for igorlabworks.com

## Under Construction

A modern landing page built with:
- Vite
- TypeScript
- React
- Radix UI
- Tailwind CSS

Features:
- Light/Dark mode toggle
- Responsive design
- Modern gradient animations

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `dist` directory, ready for deployment to GitHub Pages.
